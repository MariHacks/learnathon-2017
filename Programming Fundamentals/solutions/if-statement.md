# If Statement Solution

```python

x = input("Give me a number: ")
x = int(x)
y = input("Give me a number: ")
y = int(y)

if x > y:
    print("%i is bigger" % x)
elif y > x:
    print("%i is bigger" % y)
else:
    print("They're the same dumbo")
```
